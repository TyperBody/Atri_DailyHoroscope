# Atri_DailyHoroscope

Daily Atri Horoscope for LangBot

娱乐插件，复刻了某个机器人的运气功能

## 使用说明

通过LangBot插件市场安装后，可以通过命令`运势`获取今日运势。

还有彩蛋
